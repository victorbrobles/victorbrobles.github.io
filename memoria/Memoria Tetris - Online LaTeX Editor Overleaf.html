\section{Objetivos específicos}
\label{sec:objetivos-especificos}

Para llevar a cabo el objetivo principal es necesario también explicar otra serie de objetivos secundarios que se han llevado a cabo:\vspace{0.5cm}

\begin{itemize}
\item Aprender a desarrollar programas mediante lenguajes básicos de programación web como Javascript y HTML-5.
\item Investigar cómo se comportan estos lenguajes con la interfaz de una página web.
\item Estudiar el funcionamiento del DOM de una página web y las funcionalidades que ofrece.
\item Realizar un estudio preliminar de la tecnología A-Frame y cómo es capaz de integrarse con Javascript y HTML-5.
\item Analizar el inspector que ofrece A-Frame y cómo se puede modificar el aspecto de una aplicación a través de él.
\item Familiarizarse con cada uno de los componentes que posee A-Frame y descubrir el funcionamiento de todos ellos.
\item Entender el concepto de WebXR y comprender cómo se puede hacer que una aplicación funcione correctamente en ella.
\item Integrar una aplicación con entornos de realidad virtual y hacer que funcione tanto en versión de escritorio como con dispositivos VR (gafas y mandos).
\item Analizar las físicas que proporciona A-Frame para conseguir que los componentes interactúen entre sí y con su entorno.
\item Utilizar la biblioteca \textit{super hands} para poder manipular los componentes de A-Frame mediante los controladores VR.
\item
\end{itemize}
